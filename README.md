# Предел прочности — дело Артура Лолейта

Статический интерактивный лонгрид / digital exhibition о русском инженере
Артуре Фердинандовиче Лолейте (1868–1933) — пионере железобетона.

Чёрно-белый кинематографичный спецпроект: пиксельная типографика, моноширинный
«инженерный» текст, драматичный spotlight, музейная экспозиция.

## Структура проекта

```
predel-prochnosti/
├── index.html            ← разметка, весь текст
├── css/
│   └── style.css         ← вся стилистика и адаптив
├── js/
│   └── main.js           ← карта, before/after, сердце, форма, анимации
└── assets/
    ├── fonts/            ← Guildenstern NBP (заголовки) + JetBrains Mono (текст), self-hosted
    ├── images/           ← портреты, фигуры, 3 стадии сердца
    │   └── buildings/    ← 10 пар «тогда / сейчас»
    ├── video/            ← резерв (тизер встроен с VK через iframe)
    └── icons/            ← зарезервировано
```

## Как запустить локально

Сайт полностью статический и не требует сборки. Любой из вариантов:

```bash
# Python (есть на macOS по умолчанию)
cd predel-prochnosti
python3 -m http.server 8133
# открыть http://localhost:8133
```

Можно также просто открыть `index.html` двойным кликом, но локальный сервер
надёжнее (корректные пути и шрифты).

## Что загружать на хостинг

Загружайте **всю папку `predel-prochnosti/`** целиком как статический сайт
(точкой входа служит `index.html`). Подходит любой обычный хостинг или
GitHub Pages / Netlify / Vercel / Я.Облако — бэкенд не нужен.

## Видео-тизер

В блоке «Премьера фильма: 2028» встроен видео-тизер с VK через адаптивный `<iframe>`
(контейнер `.video-frame`, соотношение 16:9). Для воспроизведения нужен доступ
в интернет — это внешний плеер VK. Чтобы заменить ролик, поправьте `src` у `<iframe>`
внутри `<div class="video-frame">` в `index.html`.

## Технологии

Чистый HTML/CSS/JS, без зависимостей и сборщиков. Шрифты и изображения локальные.
Единственная внешняя зависимость — встроенный видео-плеер VK в блоке тизера.
Поддержаны `prefers-reduced-motion` и тач-устройства.

Заголовки набраны пиксельным шрифтом **Guildenstern NBP** (© Nate Halley, FontStruct,
лицензия CC BY-SA 3.0 — указано в футере). Основной текст — **JetBrains Mono**.

Фоновая карта Москвы (`assets/images/moscow-map.png`) сгенерирована моделью
**Google Gemini 3 Pro Image (Nano Banana Pro)** — чёрные линии улиц/колец/реки на белом фоне.
Интерактивные маркеры (1–10) расставлены по реальным географическим местам объектов
относительно Кремля, Садового кольца и излучины Москвы-реки (Егорьевск — за городом,
показан у юго-восточного края).
